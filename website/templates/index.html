Helmet Verify : AI Detection System for Safety Check

Our project leverages cutting-edge YOLOv11 AI algorithms to revolutionize motorcycle safety. We've developed a highly accurate system that detects whether a driver is wearing a helmet in real-time. If a helmet is detected, the bike's motor will start. If a helmet is not detected during riding, an alarm will sound, and the bike's motor will eventually stop, ensuring rider safety at all times.

Project Details

Project Objective

To develop a system that detects whether a driver is wearing a helmet. If a helmet is detected, the bike's motor will start. If a helmet is not detected during riding, an alarm will sound, and the bike's motor will eventually stop.

Core Features

  • Helmet Detection: The system uses computer vision techniques, including YOLOv11, to detect whether the rider is wearing a helmet.
  • Real-Time Processing: The system works with a live camera feed for real-time helmet detection, ensuring immediate response and control.

Model Configuration

  • Pre-trained Model: YOLOv11 model trained with Ultralytics for helmet detection tasks

  • Training Setup:
    • 1,200 images with YOLOv11 formatted .txt annotation files
    • Annotations created using Roboflow
    • 100 epochs of training
